This project(Smoothie Shop) serves as the hands-on application for the Snowflake Data Application Builders Workshop. In this project I built an interactive streamlit interface that allows users to customize Smoothie orders by selecting fruits from Snowflake-hosted tables. In summary:

* created UI entry forms in streamlit to collect data
* wrote python to insert collected data in snowflake tables
* made REST API calls to collect data (fetched fruit nutrition data)
* wrote python to retrieve data from snowflake table
